# 🏥 Hospital Patient Analysis Dashboard (Power BI Project)

## 📌 Project Overview

This project focuses on analyzing hospital patient data to derive meaningful insights that can support better decision-making in healthcare operations. The dashboard provides a comprehensive view of patient trends, department performance, and overall hospital activity.

---

## 🎯 Objectives

* Analyze patient admission trends over time
* Identify department-wise patient distribution
* Track key performance indicators (KPIs) related to hospital operations
* Improve data-driven decision-making through interactive dashboards

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **DAX (Data Analysis Expressions)** – Calculations and measures
* **Data Modeling** – Relationship building and schema design

---

## 📊 Key Features of Dashboard

### 🔹 1. Patient Overview

* Total number of patients
* Admission trends (daily/monthly)
* Gender and age group distribution

### 🔹 2. Department Analysis

* Department-wise patient count
* Most active departments
* Case distribution across departments

### 🔹 3. Trend Analysis

* Time-based analysis of patient admissions
* Identification of peak periods
* Seasonal patterns in hospital visits

### 🔹 4. Interactive Dashboard

* Filters for dynamic data exploration
* Drill-down capabilities
* User-friendly and clean design

---

## ⚙️ Data Processing

* Cleaned and transformed raw data
* Handled missing/null values
* Built relationships between tables
* Created calculated columns and measures using DAX
